# Research Paper Structure Analyzer Skill

🎯 **An AI-powered skill for analyzing and comparing research papers against IEEE template structure**

This skill integrates with Anthropic Claude to automatically analyze research papers and provide detailed Markdown reports with actionable recommendations.

## Features

✅ **Multi-format Support**
- PDF files
- Word documents (.docx)
- LaTeX files
- Plain text

✅ **Comprehensive Analysis**
- Extracts all paper sections
- Compares against IEEE template
- Identifies missing sections
- Evaluates section quality
- Generates compliance score

✅ **Integration with Claude**
- Use as a tool in Claude conversations
- Get detailed Markdown reports
- Actionable recommendations
